# lidar-geometric-analysis
Bu projede, TOML formatındaki 2D LIDAR verileri harici kütüphane kullanılmadan okunmuş, geçersiz ölçümler filtrelenmiş ve nokta bulutundan doğrular tespit edilmiştir. Tespit edilen doğruların kesişim noktaları ve robot merkezine olan mesafeleri hesaplanarak grafiksel olarak gösterilmiştir.
